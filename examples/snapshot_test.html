📚 API 端点

1. 原生单帧截图

GET /api/v2/snapshot/native?name={path}&fileType=stream&brightness=0&contrast=0

2. MJPEG 实时流

GET /api/v2/snapshot/mjpeg?name={path}

3. FFmpeg 截图（对比）

GET /api/v2/publish/snapshot?name={path}&fileType=stream
